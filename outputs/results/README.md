# Model Results

This folder contains model training results and evaluation metrics.

## Files (will be created in Phase 3)

- `baseline_results.json` - Metrics for baseline models (LR, RF)
- `catboost_results.json` - Metrics for CatBoost model
- `model_comparison.csv` - Comparison table of all models
- `confusion_matrix.png` - Confusion matrix visualization
- `roc_curves.png` - ROC curves for all models
- `per_lottery_performance.csv` - Performance breakdown by lottery

## Metrics Included

- F1-Score (primary metric for imbalanced data)
- Precision
- Recall
- AUC-ROC
- Confusion Matrix
- Per-class metrics

All results are generated from Phase 3 training notebooks.
